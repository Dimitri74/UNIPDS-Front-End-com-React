# Tasks CRUD App (Frontend)

Frontend em Next.js para cadastro, login e gerenciamento de tarefas com JWT.

## Documentação principal

Consulte o guia completo em [quickstart.md](quickstart.md), incluindo:

- Setup Atlas padronizado para cenário de cluster compartilhado
- Configuração de `.env.local` no frontend
- Configuração de `.env` no backend `react-tasks-crud-api`
- Endpoints, testes via Postman e diagnóstico rápido

## Execução rápida

1. Suba o backend `react-tasks-crud-api` em `http://localhost:3001`.
2. Neste projeto, execute:

```bash
npm install
npm run dev
```

3. Acesse `http://localhost:3000`.
